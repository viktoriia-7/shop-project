# React E-commerce Store

A functional online shop with product listing, shopping cart functionality, and persistent data.

## 🚀 Live Demo
[Link to your Netlify/Vercel]

## 🛠 Tech Stack
- **React**
- **Context API** (Global state for Shopping Cart)
- **LocalStorage** (Data persistence)
- **Lucide React** (Icons)

## 🔑 Key Features
- **Shopping Cart Logic:** Add/remove items, update quantity, and calculate total price in real-time.
- **Persistence:** User's cart is saved in `localStorage`, so data is not lost after page refresh.
- **Search & Sort:** Users can search products by name and sort by price.

## 🧠 What I learned
This project helped me understand global state management using **Context API**. I also learned how to synchronize React state with browser storage to improve user experience.
